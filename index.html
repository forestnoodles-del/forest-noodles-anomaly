Apps Script 部署說明

1. 打開你的 Google Sheet → 上方選單「擴充功能」→「Apps Script」

2. 刪除預設程式碼，貼上下方的 Apps Script 程式碼

4. 執行身分：「以我的身分」，存取權：「任何人」→ 部署

5. 複製產生的 URL 貼到上方「Apps Script Web App URL」欄位
